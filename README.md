# jambo - 麻雀対戦記録システム

**jambo** は、麻雀の対局記録・成績集計・賞管理を一元化するWebアプリケーションです。  
オカ・ウマ・チップ・役満賞・飛び賞などを含めた総合スコア管理が可能です。

## 主な機能
- 半荘単位でのスコア入力（オカ・ウマ自動計算）
- チップ・飛び賞・役満賞の集計
- タグ複数指定による柔軟な戦績分析
- ルールプリセット（Mリーグ、仲間内など）管理
- プレイヤーアイコン表示対応
- LINE連携による結果共有
